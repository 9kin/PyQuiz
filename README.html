Проект для Яндекс.Лицей "PyQuiz"

Что такое PyQuiz?
PyQuiz - система проведения онлайн-викторин, написанная на Python 3 с использованием библиотек PyQt, csv, socket.

Что такое викторина?
Викторина состоит из нескольких блоков, которые в свою очередь состоят из вопросов. В одном блоке на каждый вопрос дается одинаковое время, и максимальное количество очков за вопрос тоже одинаковое. На каждый вопрос может быть от 2 до 8 ответов.

Как использовать программу?
Организатор викторины запускает программу-клиент и нажимает на кнопку "Создать", выбирает уже созданную викторину или создает новую и запускает игру. После подключения к программе-серверу у организатора появится ID (или по-другому PIN) игры. Участники викторины запускают программу-клиент, нажимают кнопку "Играть" и вводят свое имя и PIN игры. Когда необходимое количество (по усмотрению организатора) игроков будет набрано, он нажимает кнопку "Старт" и игра начинается. На экране организатора отображается первый аопрос викторины и варианты ответа, раскрашенные в разные цвета. Игрокам отводится заданное организатором при создании викторины время для ответа на вопрос. Чтобы ответить на вопрос, участник должен нажать на одну из отобразившихся на его экране кнопок, того цвета, который соответствует, по его мнению правильному ответу на вопрос. Когда время, отведенное на ответ, истекает, у организатора отображается правильный ответ на вопрос, а у каждого из участников - правильность его выбора, а также его очки и позиция в рейтинге. Очки начисляются только в случае правильного ответа на вопрос, в соответствии со временем ответа - чем быстрее будет получен ответ от участника - тем больше очков будет начислено. После объяснения организатором правильного ответа участникам, организатор нажимает на кнопку "Далее" и на его экране отображается следующий вопрос. Так продолжается до тех пор, пока вопросы не закончатся. После завершения игры на экране организатора будет выведен рейтинг игроков, отсортированный по убыванию количества баллов участников.

За что отвечает каждый из файлов в проекте?
  • main.py - клиент (графический)
  • server.py - сервер (консольный)
  • config_stuff.py - инструменты для загрузки и сохранения настроек графического клиента
  • constants.py - константы
  • game_threads.py - потоки игрового процесса
  • quiz_stuff.py - классы и функции для хранения и передачи викторины
  • socket_utils.py - инструменты для работы с сетью (сокетами)
  • ui.py - интерфейс графического клиента (формы PyQT)
  • client_host.py - тестовый клиент организатора (консольный)
  • client_player.py - тестовый клиент участника (консольный)
  • config.csv - файл для сохранения настроек графического клиента
  • .quizzes - папка для сохранения викторин, созданных в графическом клиенте

Что работает на данный момент?
Полностью работает сервер.
В клиенте полностью работает создание и сохранение викторин, частично работает интерфейс участника, пока не работает интерфейс организатора.
